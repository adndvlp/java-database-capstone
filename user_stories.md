## Admin User Stories
- As an admin, I want to create and manage doctor accounts so that only authorized doctors can access the system.
- As an admin, I want to activate or deactivate user accounts so that I can control system access.
- As an admin, I want to view all appointments across the system so that I can monitor activity.
- As an admin, I want to reset user passwords so that I can assist users who are locked out.
- As an admin, I want to generate reports on appointments and usage so that I can analyze system performance.

## Patient User Stories
- As a patient, I want to register and log in so that I can access my personal dashboard.
- As a patient, I want to book an appointment with a doctor so that I can receive medical attention.
- As a patient, I want to view and cancel my upcoming appointments so that I can manage my schedule.
- As a patient, I want to view my prescriptions so that I have access to my medical history.
- As a patient, I want to receive a confirmation when booking an appointment so that I know it was registered correctly.

## Doctor User Stories
- As a doctor, I want to set my availability so that patients can only book appointments when I'm free.
- As a doctor, I want to view my scheduled appointments so that I can plan my day.
- As a doctor, I want to access a patient's record before an appointment so that I have the necessary context.
- As a doctor, I want to issue prescriptions so that patients can access them after the consultation.
- As a doctor, I want to mark an appointment as completed so that the system reflects its current status.
